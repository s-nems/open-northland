OpenNorthland

OpenNorthland is a free re-implementation of Cultures — 8th Wonder of the World. It ships no game art or sounds: on this first run it converts the assets of your own copy of the original game into its data folder. Select the folder your game is installed in (the one with Game.exe and DataX) to begin.
